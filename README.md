## overview
This project demonstrates a complete **end-to-end data analytics workflow**, from raw data processing to business insights visualization.
The goal is to analyze customer purchase behavior, identify spending trends, segment customers, and uncover factors driving revenue and product preferences.

By combining **Python, SQL, and Power BI**, this project shows practical data analysis skills for solving real business problems and communicating results effectively.

## ğŸ” Steps Performed

### 1. Data Loading

* Imported CSV file into Python using Pandas
* Previewed data structure and basic statistics

### 2. Data Cleaning & Preparation

* Handled missing and duplicate values
* Standardized column names and data types
* Performed feature engineering (e.g., total spend, customer tenure)

### 3. Exploratory Data Analysis (EDA)

* Analyzed sales by region, gender, and age groups
* Identified top products and revenue contributors
* Visualized purchase frequency and seasonal trends

### 4. SQL Analysis

* Loaded cleaned data into **PostgreSQL/MySQL/SQL Server**
* Executed SQL queries to answer key business questions:

  * Top 10 customers by revenue
  * Average order value per region
  * Discount utilization patterns
  * Monthly and yearly sales trends

### 5. Dashboard Creation (Power BI)

* Built interactive dashboard with filters for customer segments, categories, and time period
* Displayed KPIs: Total Revenue, Avg Order Value, Active Customers, Discount Impact
